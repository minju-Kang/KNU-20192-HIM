Information Sharing Test Plan
:
- 공유 신청 카테고리를 클릭했을 때 신청창이 제대로 뜨는지 확인한다.
- 신청창에 넣은 값이 제대로 입력되는지 확인한다.
- 공유하고 싶은 상대방의 ID가 Patient Manegement에 존재하는지 확인한다.
- 사용자가 '신청' 버튼 클릭 시 상대방에게 수락 요청 메시지가 전송되는 지 확인한다.
- 상대방이 '수락' 버튼 클릭 시 서로의 데이터가 공유되는지 확인한다.
- 상대방이 '거절' 버튼 클릭 시 서로의 데이터가 공유되지 않는 것을 확인한다.


Collective Statistical Data Test Plan
:
- 의료데이터 보기 카테고리를 클릭했을 때 검색창이 제대로 뜨는지 확인한다.
- 검색창에 넣은 값이 제대로 입력되는지 확인한다.
- 검색하고 싶은 질병의 이름이 Disease에 존재하는 지 확인한다.
- 사용자가 '보기' 버튼 클릭 시 Patient Management에 그 질병을 가진 환자가 있는 지 확인한다.
- 질병을 가진 환자들이 있다면 사용자에게 질병을 가진 환자 list를 제공하는 지 확인한다.
